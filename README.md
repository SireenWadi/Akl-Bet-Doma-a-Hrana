## ✨ About the Project

**Akl Bet** (أكل بيت) is a luxury single-page restaurant website for an authentic Palestinian home kitchen based in Subotica, Serbia. Built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

> *"From our kitchen to your table — the finest fresh ingredients for an unforgettable dining experience."*

## 🚀 Features

- 🌍 **Trilingual** — Arabic (RTL), Serbian & English with seamless switching
- 🎨 **Luxury Dark Theme** — Gold & cream palette with grain overlay
- 📱 **Fully Responsive** — Mobile-first design
- 🍽️ **Interactive Menu** — Filter by Main Dishes, Mezze, Pastries, Sweets
- 🖼️ **Gallery Lightbox** — Packaging & food photography
- 💬 **WhatsApp Integration** — Direct ordering from every dish card
- ❓ **FAQ Accordion** — Smooth animated answers
- ✨ **Scroll Animations** — Staggered reveal on every card
- 🎁 **Special Offer Banner** — Animated shimmer effect
- 📢 **Announcement Bar** — Smooth marquee scroll

## 🛠️ Tech Stack

```

CSS3        → Custom animations, CSS Variables, Grid & Flexbox
JavaScript  → Language switching, menu filtering, FAQ accordion, lightbox
Google Fonts → Amiri · Playfair Display · Cormorant Garamond · Noto Sans Arabic
```
